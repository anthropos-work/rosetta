## Adversarial review (close Phase 2c, 2026-06-09)

Scenarios considered against the set-dress chain (`up-injected.sh`) + the shared engine (`dev-setdress.sh`). Every one was already handled by the build/harden; recorded here so future reviewers see what was probed.

1. **Non-numeric N reaches the `$((5432 + N * 10000))` arithmetic** (the M18 FINDING-A1 class). Handled: `dev-setdress.sh:71` validates `N` against `^[0-9]+$` and `die`s *before* any arithmetic or CLI call — pinned by `test_non_integer_n_is_rejected_before_any_cli` + `test_guard_rejects_non_integer`. The demo offset comes from the registry-resolved `$OFFSET`, not user free-text.
2. **A trailing value-flag (`--dsn` / `--seed` / `--stack-type`) with no argument under `set -u`** would `die` with a raw `$2: unbound variable`. Handled: the `needval` guard prints a clear "option needs a value" — pinned by `test_trailing_value_flag_dies_with_a_clear_message` + `test_trailing_stack_type_flag_dies_with_a_clear_message`.
3. **A demo bring-up at N=0** vs the n=0-dev guard. By design `up-injected.sh` always passes `--force`, so the dev-only safety net never blocks a demo (demos start at N≥1). The guard still fires for an *un-forced* N=0 of *either* type — pinned by `test_n0_guard_fires_for_demo_type_too` + `test_force_keeps_the_engine_guard_from_blocking_a_demo`. The net stays a dev guard; demo mode does not weaken it.
4. **A future maintainer forks a demo-only set-dress copy** (drift from the dev engine). Handled by the convergence fence `test_reuses_the_dev_setdress_engine_via_stack_type_demo` — it fails the moment the demo chain stops calling `../dev-stack/dev-setdress.sh --stack-type demo`.
5. **A `stacksnap capture` slips onto a bring-up path** (the load-bearing prod-safety regression). Handled by `test_capture_is_never_invoked_on_a_bring_up` (the chain body) + `test_capture_never_runs_even_on_the_cache_miss_degraded_path` (the engine's degraded branch — harden's extension). Mutation-pinned: adding a capture fallback to the cache-miss branch fails the test.

No scenario revealed an unhandled risk; no fix needed. (Question asked, answer recorded.)
