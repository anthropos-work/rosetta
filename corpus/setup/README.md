# Setup Documentation

This directory contains all documentation related to setting up the Anthropos development environment.

## Files

*   **[setup_guide.md](./setup_guide.md)**: Comprehensive step-by-step guide for setting up the Anthropos platform on a fresh machine. Covers both macOS and Linux environments.

*   **[setup_checklist_macos.md](./setup_checklist_macos.md)**: macOS-specific checklist for tracking setup progress. Copy this to your workspace to track your setup status and log errors.

*   **[setup_checklist_linux.md](./setup_checklist_linux.md)**: Linux-specific checklist for tracking setup progress. Copy this to your workspace to track your setup status and log errors.

*   **[setup_checklist.md](./setup_checklist.md)**: Generic checklist (deprecated in favor of OS-specific versions above).

## Quick Start

1.  Read the **[Setup Guide](./setup_guide.md)** for detailed instructions.
2.  Copy the appropriate checklist ([macOS](./setup_checklist_macos.md) or [Linux](./setup_checklist_linux.md)) to your `anthropos-dev/` directory.
3.  Follow the guide and check off items as you complete them.

## For Maintainers

When updating setup documentation:
*   Update the **setup_guide.md** with new steps or changes.
*   Update **both** OS-specific checklists to reflect the changes.
*   Keep checklists lean—they're for tracking, not detailed instructions.
